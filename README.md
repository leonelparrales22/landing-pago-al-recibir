# Landing Page - Pago al Recibir

Landing page estática para validar la hipótesis de crecimiento de una nueva opción de pago en Mercado Libre: pago por transferencia bancaria o código QR al recibir (contra entrega digital).

## Características

- Diseño moderno y responsive
- Colores corporativos (azul oscuro #1e3a8a y naranja #f97316)
- Formulario de captura de correos electrónicos
- Prueba social estática
- Sin backend, todo en un solo archivo HTML

## Cómo Ejecutar Localmente

1. Descarga o clona este repositorio
2. Navega al directorio del proyecto
3. Haz doble clic en el archivo `index.html` para abrirlo en tu navegador web

No requiere instalación de dependencias ni servidor web.

## Cómo Publicar en GitHub Pages

1. Sube este repositorio a GitHub
2. Ve a la configuración del repositorio (Settings)
3. En la sección lateral, haz clic en "Pages"
4. En "Source", selecciona "Deploy from a branch"
5. En "Branch", selecciona `main` (o `master`) y `/ (root)`
6. Haz clic en "Save"
7. Espera unos minutos y GitHub Pages te proporcionará una URL pública
